
# 📚 BS Data Science - PUCIT 

Welcome to the **BS_DATA_SCIENCE_PUCIT** repository! 🎓 This repository is created to **store and share study materials** for the **BS Data Science program at PUCIT**. It contains essential resources such as:

✅ **Lecture Slides** 📄  
✅ **Books & Reference Materials** 📚  
✅ **Quizzes & Past Papers** ✍️  
✅ **Assignments & Semester Projects** 📝  
✅ **Handwritten & Digital Notes** 🗒️  

The goal of this repository is to **help students access organized and structured learning materials** to excel in their coursework, exams, and projects. If you're a **BS Data Science student**, this repository will be your one-stop destination for all study-related materials.  

---
